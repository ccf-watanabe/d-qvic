SNS利用規約

はじめに

株式会社データ・キュービックは、所有するソーシャルメディア公式アカウントの運営に関して、次の通りご利用規約（以下、「本規約」といいます）を定めています。

株式会社データ・キュービックが運営するソーシャルメディアアカウント

  • 公式Facebookページ「ここいろ」

    URL : https://www.facebook.com/cocoiro.dqvic

  • 公式Instagramビジネスプロフィール「ここいろ」

    URL : https://www.instagram.com/cocoiro.dqvic

本規約の適用範囲・変更

  • 本規約は、株式会社データ・キュービック（以下、「当社」といいます）が運営する公式アカウント（以下、「当アカウント」といいます）をご利用されるすべてのみなさまに適用されるものとします。
  • ご利用者のみなさまが当アカウントをご利用になる場合、本規約を承諾されたものとみなします。
  • 当社は、ご利用のみなさまの事前の承諾および事後の通知等を要することなく、本規約を変更できるものとします。変更後は当社のウェブサイト上に掲載した時点より、効力を生じるものとします。

投稿・コメントへの回答について

ご利用のみなさまから投稿されたコメントにつきましては、業務等の参考にさせていただきますが、原則として、当社から個別の回答はいたしません。ご意見・ご質問・ご要望につきましては、当社へお問い合わせください。

基本情報へのアクセス

当アカウントのファンおよびフォロワーとしてご登録いただいた場合は、ご利用者のお名前やプロフィール、写真、性別等、ご利用者がアカウント設定上公開しているすべての基本情報へ、当アカウントからのアクセスを許諾されたものとみなします。
また、イベントやキャンペーンの円滑な遂行のため、ダイレクトメッセージを送信させていただく場合がございます。

禁止行為

当社は当アカウントの利用にあたり、以下に該当もしくは該当する恐れのある行為を禁止します。禁止行為にあたる投稿・コメントは予告なく削除することがあります。

  • 第三者または当社に迷惑、不利益もしくは損害を与える行為
  • 第三者または当社の商品やサービスを誹謗中傷する行為
  • 第三者または当社を誹謗中傷し、第三者または当社の名誉を毀損する行為
  • 当社を含む第三者になりすます行為
  • 第三者または当社の商標権、著作権そのた知的財産権、プライバシーその他の権利を侵害する行為
  • 政治活動、選挙活動、宗教活動またはこれらに類似する行為
  • 金利・為替・株式市場等のマーケットの動向について断定的判断を提供する行為
  • 法律や法令、公序良俗に反する行為
  • 犯罪に結びつく行為、または恐れのある行為
  • アフィリエイト、広告、宣伝目的の内容が含まれる投稿やデータ送信行為
  • Facebook／Instagramの利用規約に違反する行為
  • その他、当社が不適切と判断する行為

免責事項

  • 当アカウントに当社が掲載する情報は、情報提供を主な目的としています。
  • 当アカウントで当社が発信する情報は、必ずしも当社の公式発表・見解を表すものではございません。当社の公式発表・見解は、当社のホームページで行います。
  • 当社は、当アカウント上に公開されている情報を予告なく変更・削除することがあります。また、やむをえない事由によりご利用者のみなさまに予告なく当アカウントの公開を中断あるいは中止する場合があります。

  • 当社は、ご利用者によって投稿された情報について一切責任を負いません。利用者間、またはご利用者と第三者の間でトラブルまたは紛争が生じた場合、ご利用者が自己の責任において当該トラブルまたは紛争を解決するものとし、当社は一切責任を負いません。
  • ご利用者が当社に対し、損害賠償請求権等の何らかの請求権を有する場合、その発生原因が生じた日から1年以内に権利行使をしなければ、当該権利は失効するものとします。

準拠法・管轄
